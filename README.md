# ⏱️ 工時計算器 (Work Hour Record)

一個輕量、現代且功能強大的工時與收入紀錄網頁應用程式 (Web App)。專為需要精確掌握工作時間、薪資計算及額外獎金收入的使用者（如業務人員、自由接案者）設計。

此專案由 **Trae** (AI Editor) 協作開發，完美支援電腦與手機瀏覽器，並可輕鬆嵌入 **Google 協作平台 (Google Sites)**。

---

## ✨ 核心功能

### 1. ⏱️ 智慧打卡系統
*   **快速打卡**：一鍵「上班/下班」打卡，自動記錄精確時間。
*   **休息扣除**：支援設定預設休息時間，打卡時若時長達標，系統會主動詢問是否自動扣除。
*   **手動補登**：忘記打卡也沒關係，支援手動輸入日期與上下班時間，且每筆紀錄可獨立自訂休息時長。

### 2. 📊 多維度統計看板 (自動連動)
看板會即時顯示以下資訊，幫助您掌握財務狀況：
*   **今日時數**：即時掌握當天已累積的工作時數。
*   **上個月統計**：**[自動更新]** 每月 1 號自動滾動，顯示上個月的總時數與「薪資 + 獎金 = 總收入」。
*   **本月累積**：累計當月至今的所有工時。
*   **區間統計/總收入**：配合篩選器，顯示特定期間的總時數與詳細收入結構。
*   **預估薪資**：根據設定的時薪，自動計算本月的純工時計薪。
*   **獎金/其他**：顯示本月份在備忘錄中記錄的所有額外獎金總和。
*   **本月總收入**：自動加總「時薪收入」與「獎金收入」。

### 3. 💡 獎金備忘錄 (不計工時收入)
*   **專為業務設計**：可獨立記錄如「賣車獎金」、「業績抽成」等不屬於工時計算的收入。
*   **明細列表**：每筆獎金皆可備註並標註日期，方便後續對帳。
*   **範圍篩選**：**[新功能]** 備忘錄區塊也支援下拉選單，可切換查看「完整記錄」、「本月紀錄」或「上個月紀錄」，並自動加總該範圍的獎金。
*   **一鍵管理**：支援單筆刪除或一鍵清空所有備忘錄。

### 4. 🔍 紀錄篩選與管理
*   **快速選單**：下拉選單可切換「完整記錄」、「本月紀錄」、「上個月紀錄」。
*   **自定義範圍**：可自由選擇起始與結束日期，精確查看特定專案或週期的紀錄。
*   **Excel 匯出**：一鍵將所有紀錄匯出為 `.csv` 檔，方便使用 Excel 進行二次加工。

### 5. ☁️ 安全備份與同步
*   **LocalStorage**：資料預設儲存於您的瀏覽器，無須註冊，隱私安全。
*   **檔案備份**：支援下載 `.json` 備份檔，輕鬆在不同電腦間轉移資料。
*   **手機文字備份**：專為 LINE 等內建瀏覽器優化，透過「複製代碼」或「一鍵分享」即可完成手機資料轉移。
*   **雲端同步**：支援連結個人 Google Sheets，實現真正的跨裝置雲端資料保存。

## 🚀 操作指南

### 設定您的計薪標準
1. 點擊右上角 `⚙️ 設定`。
2. 輸入您的「時薪」與「預設休息時間」。
3. 點擊「儲存設定」，看板將立即依照新標準重新計算。

### 如何查看不同月份的紀錄？
1. 到頁面下方的「工時紀錄」區塊。
2. 點擊「顯示範圍」下拉選單。
3. 選擇「本月紀錄」或「上個月紀錄」，上方的**區間統計卡片**會隨之更新顯示該月的詳細收入結構。

### 如何修正錯誤紀錄？
*   **修改休息時間**：使用「🛠️ 補登/修改休息時間」功能，選擇日期後輸入新的分鐘數即可批量更新該日紀錄。
*   **修改時間/日期**：若時間記錯，請先刪除該筆紀錄，再使用「📝 手動新增紀錄」重新補登正確的時間。

## 🛠️ 專案結構與技術
*   **index.html**: 響應式 UI 介面設計。
*   **style.css**: 現代化樣式與手機版優化。
*   **script.js**: 包含自動計算、篩選邏輯與雲端同步的核心引擎。
*   **技術棧**: Vanilla JavaScript, CSS Grid/Flexbox, Google Sheets API v4.
